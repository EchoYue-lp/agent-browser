# 文档

欢迎使用 Agent Browser 文档！

## 概述

Agent Browser 是专为 AI Agent 设计的浏览器自动化工具集。支持多种协议（MCP、HTTP），通过 Accessibility Tree 提供语义化的元素定位能力。
